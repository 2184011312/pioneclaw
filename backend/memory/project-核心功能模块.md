## 核心功能模块

- **Skill 系统**: 用户可创建/编辑/配置技能，支持提交评审，支持 YAML inline 两种格式
- **多 Provider 支持**: OpenAI / Anthropic / DeepSeek 等多家 LLM 提供商
- **权限管理**: 用户级 / 组织级 / 系统级三级权限模型
- **WebSocket 实时推送**: Agent 思考过程实时流式输出
- **工具系统**: 内置 30+ 工具，支持搜索、文件读写、代码执行等
