## 1. 登录系统

### 访问地址

- 前端地址：`http://localhost:5173`（开发环境）
- 后端 API：`http://localhost:8000/docs`

### 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| `admin` | `admin123` | 超级管理员 |

### 登录步骤

1. 打开浏览器访问系统地址
2. 输入用户名和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转到仪表盘

### 忘记密码

请联系系统管理员重置密码。

## 2. 仪表盘 (Dashboard)

仪表盘是系统的首页，展示系统整体运行状态和关键指标。

### 功能概览

#### 欢迎区域
- 显示当前用户名称和问候语
- 快速进入「智能对话」的按钮

#### 概览指标卡片（4个）

| 卡片 | 说明 |
|------|------|
| **中心状态** | 显示系统是否在线运行 |
| **在线智能体** | 当前活跃的 AI 智能体数量 |
| **今日任务** | 今天创建的任务总数 |
| **API 调用** | 今日 API 调用次数 |

#### API 用量统计

展示详细的 API 使用情况：
- **调用次数**：总 API 调用数
- **总 Tokens**：消耗的 Token 总量
- **输入 Tokens**：用户输入消耗
- **输出 Tokens**：AI 回复消耗
- **平均延迟**：响应时间（毫秒）
- **失败次数**：调用失败的数量

#### 模型分布

以进度条形式展示各模型的使用占比。

#### 任务分布

以彩色条形图展示今日任务状态：
- 🟢 已完成（绿色）
- 🟠 进行中（橙色）
- 🔵 待办（蓝色）
- ⚪ 已取消（灰色）

#### 最近日志

展示最近的 API 调用记录：
- 绿点：调用成功
- 红点：调用失败
- 显示模型名称、Token 数、响应时间

### 操作说明

| 操作 | 说明 |
|------|------|
| 点击「立即对话」 | 跳转到智能对话页面 |
| 点击「查看全部」 | 跳转到日志管理页面 |
| 页面自动刷新 | 每 30 秒自动更新数据 |

## 3. 智能对话 (Chat)

智能对话是系统的核心功能，提供与 AI 的交互界面。

### 界面布局

```
┌─────────────────────────────────────────────────┐
│  会话列表  │        对话区域                    │
│  (左侧)    │   ┌─────────────────────────────┐  │
│            │   │  消息气泡                    │  │
│  [新建对话] │   │  用户消息 (右侧蓝色)         │  │
│  [对话1]   │   │  AI 回复 (左侧灰色)          │  │
│  [对话2]   │   │  工具调用 (可展开)           │  │
│  [对话3]   │   └─────────────────────────────┘  │
│            │   ┌─────────────────────────────┐  │
│            │   │  输入框 + 发送按钮           │  │
│            │   │  [快速/思考] [附件] [发送]    │  │
│            │   └─────────────────────────────┘  │
└─────────────────────────────────────────────────┘
```

### 会话列表（左侧）

| 操作 | 说明 |
|------|------|
| 点击「+」按钮 | 创建新对话 |
| 点击对话项 | 切换到该对话 |
| 点击折叠按钮 | 收起侧边栏（只显示图标） |

### 对话区域（右侧）

#### 顶部工具栏

| 按钮 | 功能 |
|------|------|
| 模型选择下拉框 | 选择使用的 AI 模型 |
| 更多操作（⋯） | 清空对话、导出对话、删除对话 |

#### 消息显示

- **用户消息**：蓝色气泡，显示在右侧
- **AI 回复**：灰色气泡，显示在左侧
- **工具调用**：可展开/收起的卡片，显示工具名称和执行结果

#### 消息元数据

AI 回复下方显示：
- ⏱ 响应延迟（毫秒）
- ↑ 输入 Token 数
- ↓ 输出 Token 数

#### 快捷操作

在 AI 回复上悬停显示：
- 📋 复制：复制消息内容
- 🔄 重新生成：重新请求 AI 回复

### 输入区域

#### 模式切换

| 模式 | 说明 |
|------|------|
| **快速模式** | 直接调用模型，不使用工具 |
| **思考模式** | 启用 ReAct 循环，AI 会自动调用工具完成任务 |

#### 其他功能

| 按钮 | 功能 |
|------|------|
| 📎 附件 | 上传文件（开发中） |
| ⏹ 停止 | 停止当前生成（生成中显示） |
| 发送 | 发送消息（快捷键：Enter） |

### 斜杠命令

在输入框输入 `/` 可使用快捷命令：

| 命令 | 功能 |
|------|------|
| `/new` | 新建对话 |
| `/clear` | 清空当前对话 |
| `/history` | 查看历史对话 |
| `/model` | 查看可用模型 |
| `/export` | 导出当前对话 |
| `/help` | 显示帮助信息 |

### 工具执行状态

当 AI 调用工具时，右下角显示实时状态：
- 🔧 运行中：显示进度条
- ✅ 完成：显示执行时间
- ❌ 失败：显示错误信息

## 4. 智能体管理 (Agents)

智能体（Agent）是配置了特定行为和能力的 AI 实体。

### 功能说明

| 功能 | 说明 |
|------|------|
| 创建智能体 | 新建一个 AI 智能体配置 |
| 编辑智能体 | 修改智能体的名称、模型、提示词等 |
| 启用/禁用 | 切换智能体的激活状态 |
| 删除智能体 | 删除智能体配置 |

### 列表字段

| 字段 | 说明 |
|------|------|
| 名称 | 智能体的唯一标识符 |
| 显示名称 | 智能体的友好名称 |
| 描述 | 智能体的功能描述 |
| 模型 | 使用的 AI 模型 |
| 轮次 | 最大对话轮次 |
| 状态 | 激活/未激活 |

### 创建/编辑智能体

#### 表单字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 唯一标识符，如 `code-assistant` |
| 显示名称 | ✅ | 友好名称，如「代码助手」 |
| 描述 | ❌ | 功能描述 |
| 模型 | ✅ | 从已配置的模型中选择 |
| 最大轮次 | ❌ | 对话最大轮次，默认 20 |
| 系统提示词 | ❌ | 定义智能体行为的系统指令 |

#### 系统提示词示例

```
你是一个专业的代码助手，擅长：
- 代码编写和调试
- 代码审查和优化
- 技术问题解答

请用清晰、简洁的方式回答问题。
```

### 操作按钮

| 按钮 | 功能 | 颜色 |
|------|------|------|
| 编辑 | 修改智能体配置 | 蓝色 |
| 启用/禁用 | 切换状态 | 橙色 |
| 删除 | 删除智能体 | 红色 |

### 功能说明

| 功能 | 说明 |
|------|------|
| 创建技能 | 新建自定义技能 |
| 编辑技能 | 修改技能内容和配置 |
| 配置技能 | 设置技能的参数 |
| 热重载 | 重新加载技能文件 |

### 统计卡片

| 卡片 | 说明 |
|------|------|
| 总数 | 技能总数 |
| 激活数 | 已启用的技能数 |
| 已配置 | 完成配置的技能数 |
| 有模式 | 有配置模式的技能数 |

### 列表字段

| 字段 | 说明 |
|------|------|
| 名称 | 技能唯一标识 |
| 标签 | AUTO（自动激活）、YAML（格式）、来源 |
| 描述 | 技能功能说明 |
| 状态 | 开关切换启用/禁用 |
| 配置 | 配置状态（已配置/未配置/缺少字段） |
| 依赖 | 检查依赖是否满足 |

### 创建技能

#### 表单字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 唯一标识符 |
| 显示名称 | ❌ | 友好名称 |
| 描述 | ❌ | 功能描述 |
| 标签 | ❌ | 分类标签（auto、tool、knowledge、workflow） |
| 自动加载 | ❌ | 是否自动被智能体加载 |

### 技能配置

点击「配置」按钮可设置技能参数：
- 文本参数
- 密码参数（API Key 等）
- 数字参数
- 布尔开关
- 下拉选择

### 模式管理

点击「模式管理」可创建技能配置模式：
1. 输入技能名称
2. 定义版本号
3. 编写 JSON 格式的字段定义
4. 点击创建

### 热重载

点击「热重载」按钮可重新加载技能文件，无需重启服务。

## 7. 定时任务 (Cron)

定时任务用于周期性执行智能体任务。

### 功能说明

| 功能 | 说明 |
|------|------|
| 创建任务 | 新建定时任务 |
| 编辑任务 | 修改任务配置 |
| 启用/禁用 | 切换任务状态 |
| 立即执行 | 手动触发一次执行 |
| 删除任务 | 删除定时任务 |

### 列表字段

| 字段 | 说明 |
|------|------|
| ID | 任务编号 |
| 名称 | 任务名称 |
| Cron 表达式 | 定时规则 |
| 调度说明 | 中文描述执行时间 |
| 描述 | 任务描述 |
| 状态 | 激活/未激活 |
| 执行次数 | 已执行次数 |
| 上次执行 | 上次执行时间 |

### Cron 表达式格式

```
┌───────────── 分钟 (0-59)
│ ┌───────────── 小时 (0-23)
│ │ ┌───────────── 日期 (1-31)
│ │ │ ┌───────────── 月份 (1-12)
│ │ │ │ ┌───────────── 星期 (0-6, 0=周日)
│ │ │ │ │
* * * * *
```

### 常用示例

| 表达式 | 说明 |
|--------|------|
| `0 9 * * *` | 每天上午 9:00 |
| `0 */2 * * *` | 每 2 小时 |
| `30 8 * * 1-5` | 工作日 8:30 |
| `0 0 * * 0` | 每周日 0:00 |
| `0 0 1 * *` | 每月 1 日 0:00 |

### 创建定时任务

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 任务名称 |
| Cron 表达式 | ✅ | 定时规则 |
| 关联智能体 | ❌ | 要执行的智能体 |
| 输入参数 | ❌ | JSON 格式的参数 |
| 描述 | ❌ | 任务描述 |
| 状态 | ❌ | 是否立即启用 |

## 8. 记忆系统

PioneClaw 提供三种记忆系统，帮助 AI 保持上下文和知识。

### 8.1 长期记忆 (Memories)

存储用户级别的持久化记忆。

#### 功能

| 操作 | 说明 |
|------|------|
| 添加记忆 | 手动添加一条记忆 |
| 搜索记忆 | 按关键词搜索记忆 |
| 编辑记忆 | 修改记忆内容 |
| 删除记忆 | 删除指定记忆 |

#### 记忆字段

| 字段 | 说明 |
|------|------|
| 内容 | 记忆的具体内容 |
| 类型 | 事实、偏好、事件等 |
| 重要性 | 1-5 分 |
| 创建时间 | 记忆创建时间 |

### 8.2 智能体记忆 (Agent Memories)

存储智能体级别的记忆。

#### 功能

- 查看各智能体的记忆
- 按智能体筛选记忆
- 管理智能体的知识库

### 8.3 分层记忆 (Layered Memory)

分层记忆系统，按重要性分级存储。

#### 层级结构

| 层级 | 说明 | 保留时间 |
|------|------|----------|
| 工作记忆 | 当前对话上下文 | 会话期间 |

| 长期记忆 | 持久化知识 | 永久 |
| 压缩记忆 | 摘要和压缩信息 | 永久 |

## 常见问题

### Q: 如何添加新的 AI 模型？

1. 进入「AI 管理」→「AI 配置」
2. 点击「创建」按钮
3. 填写模型信息（提供商、模型 ID、API Key 等）
4. 点击「测试连接」验证配置
5. 保存配置

### Q: 如何创建自定义技能？

1. 进入「技能管理」页面
2. 点击「创建」按钮
3. 填写技能名称和内容
4. 如需配置参数，点击「模式管理」创建配置模式
5. 启用技能

### Q: 如何设置定时任务？

1. 进入「定时任务」页面
2. 点击「创建」按钮
3. 填写任务名称和 Cron 表达式
4. 选择要执行的智能体
5. 启用任务

### Q: 如何添加新用户？

1. 进入「用户与权限」→「用户」
2. 点击「创建用户」按钮
3. 填写用户信息
4. 分配角色和组织
5. 保存后用户可使用账号登录
